# Visao Geral da Disciplina

## Identificacao rapida

- Disciplina: Desenvolvimento para Dispositivos Moveis
- Curso: Tecnologia em Sistemas para Internet (TSI)
- Carga horaria: 60h
- Formato: 80 aulas de 45 minutos
- Organizacao: 40 encontros (2 aulas por encontro)
- Abordagem: React Native (multiplataforma)

## Proposito da disciplina

Capacitar o estudante para desenvolver aplicativos moveis multiplataforma, com foco em interfaces, navegacao entre telas, persistencia local, integracao com servicos web e uso de recursos embarcados.

## Ementa (resumo)

- Tecnicas de programacao para redes sem fio e redes celular.
- Programacao usando tecnologias para dispositivos moveis.

## Objetivo geral

- Programar em ambiente movel embarcado.

## Competencias desenvolvidas

1. Configurar e operar ambiente de desenvolvimento React Native.
2. Construir interfaces moveis com formularios e listas.
3. Implementar navegacao em aplicativos multijanelas.
4. Persistir dados localmente no dispositivo.
5. Consumir APIs e servicos web.
6. Integrar recursos embarcados (camera, localizacao, bluetooth).
7. Entregar aplicativo funcional com qualidade tecnica minima.

## Estrutura por unidades

### Unidade 1 (Encontros 1 a 14)

Fundamentos de desenvolvimento movel, interfaces (formularios e listas) e navegacao entre telas.

### Unidade 2 (Encontros 15 a 27)

Persistencia local, consumo de servicos web e integracao com recursos embarcados.

### Unidade 3 (Encontros 28 a 40)

Multiplataforma, qualidade (testes/performance/build) e projeto integrador final.

## Avaliacao (resumo)

- U1: 30 pontos
- U2: 30 pontos
- U3: 40 pontos
- Total: 100 pontos

Ver detalhes em `docs/avaliacao.md`.

## Produtos finais esperados

- Aplicativos praticos por unidade.
- Projeto integrador com demonstracao funcional.
- Repositorio com codigo organizado e documentado.
