# Encontro 35 - Sprint 1 do projeto integrador

## Objetivos

- Iniciar implementação do núcleo do projeto.
- Validar estrutura, navegação e primeira funcionalidade principal.
- Estabelecer rotina de acompanhamento.

## Sugestão de condução

1. Revisão rápida do escopo.
2. Organização das tarefas da sprint.
3. Desenvolvimento assistido.
4. Checkpoint de demonstração curta.

## Entregáveis mínimos

- repositório organizado;
- navegação funcionando;
- uma funcionalidade de ponta a ponta.

## Atividade

- Cada equipe demonstra o fluxo principal já operacional.
- Professor registra bloqueios técnicos para intervenção dirigida.
