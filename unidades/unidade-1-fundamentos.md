# Unidade 1 - Fundamentos, Interfaces, Navegação e Dados

## Período

Encontros 1 a 20

## Competências

- Configurar ambiente de desenvolvimento React Native.
- Construir interfaces com formulários e listas.
- Implementar navegação multijanelas.
- Persistir e recuperar dados localmente.
- Consumir serviços web com tratamento de estados de requisição.

## Conteúdos

1. Introdução à programação móvel e ecossistema React Native.
2. Programação de interfaces.
   1. Estilo formulário.
   2. Formato de listas.
3. Técnicas de navegação em aplicativos multijanelas.
4. Persistência local no dispositivo (AsyncStorage e SQLite).
5. Consumo de serviços web e tratamento de loading/erro.

## Produtos de aprendizagem

- App com múltiplas telas.
- Cadastro via formulário.
- Listagem dinâmica de dados.
- App com persistência local.
- App consumindo API REST com feedback de estado.
