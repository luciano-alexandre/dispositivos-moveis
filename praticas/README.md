# Práticas da Disciplina

Este diretório organiza as práticas por unidade.

Sugestão de subpastas:

- `u1-praticas/`: interfaces e navegação
- `u2-praticas/`: persistência, API e recursos embarcados
- `u3-praticas/`: testes, performance e preparação de release

Sugestão de padrão por prática:

- `README.md` com objetivo, requisitos e critério.
- `starter/` com código inicial.
- `solucao/` (opcional para docente).
