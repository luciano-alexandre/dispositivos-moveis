# Praticas da Disciplina

Este diretorio organiza as praticas por unidade.

Sugestao de subpastas:

- `u1-praticas/`: interfaces e navegacao
- `u2-praticas/`: persistencia, API e recursos embarcados
- `u3-praticas/`: testes, performance e preparacao de release

Sugestao de padrao por pratica:

- `README.md` com objetivo, requisitos e criterio.
- `starter/` com codigo inicial.
- `solucao/` (opcional para docente).
